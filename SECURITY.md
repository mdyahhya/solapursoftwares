# Security Policy

## Supported Versions

Only the current main deployment of the website is supported for security updates.

| Version | Supported |
| ------- | --------- |
| v1.x.x  | Yes       |

## Reporting a Vulnerability

We take the security and privacy of this directory very seriously. If you discover a vulnerability or have security concerns, please report them responsibly.

**Do not open a public GitHub issue for security vulnerabilities.**

Instead, please send an email to the repository owner or submit a private report if configured on GitHub. We will acknowledge receipt of your report within 48 hours and work with you to analyze and coordinate the release of a fix before public disclosure.

Thank you for helping keep our directory and users secure!
